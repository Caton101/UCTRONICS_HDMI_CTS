# UC586 
## Add 480x800 resolution
```bash 
wget https://raw.githubusercontent.com/UCTRONICS/UCTRONICS_HDMI_CTS/master/uc586/hdmi_480x800_cfg.sh
```
```bash
sudo chmod +x hdmi_480x800_cfg.sh
```
```bash
sudo ./hdmi_480x800_cfg.sh
```
